# CS546 Human Autonomy Benchmark

A benchmark suite for evaluating human-AI collaboration and autonomy trade-offs.

## Quick Start

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run experiments: `python generate_model_answers_<model_name>.py`

## Structure

- `/experiments` - Benchmark tasks and evaluation metrics
- `/models` - AI model implementations
- `/data` - Dataset and evaluation scripts
